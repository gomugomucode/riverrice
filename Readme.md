# RiverRice

Minimal Wayland rice using the River compositor.

## Features

- River (dynamic tiling)
- Foot (terminal)
- Fuzzel (launcher)
- Waybar (status bar)
- Zathura (PDF viewer)
- Lf (file manager)
- Waylock (screen locker)
- Tmux, Fastfetch, minimal scripts

## Install

```bash
git clone https://github.com/gomugomucode/riverrice ~/riverrice
cd ~/riverrice
chmod +x scripts/install.sh
./scripts/install.sh
